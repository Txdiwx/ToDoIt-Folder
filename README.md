# 📝 To-Do List Web Application

## 📌 Project Overview
This is a simple To-Do List web application project built for IU's " Portfolio Project: Java and Web Development (DLBCSPJWD01)" course that allows users to create, view, update, and delete tasks.  
The application uses **HTML, CSS, JavaScript** for the frontend and **Node.js, Express, SQLite** for the backend.

---

## 🚀 Features
✔ Login and registration
✔ Add a new task  
✔ View all tasks dynamically  
✔ Mark tasks as completed  
✔ Delete tasks  
✔ Persistent storage using SQLite database  
✔ Responsive UI for desktop and mobile devices  

---
## 🛠️ Tech Stack
Backend: Node.js, Express
Database: MongoDB (via Mongoose)
Frontend: HTML, CSS, JavaScript
